# 🎓 Student Management System

A modern and responsive **Student Management System** built with React.js.

This project demonstrates complete **CRUD operations (Create, Read, Update, Delete)** using Axios for API communication and Tailwind CSS for a clean and responsive user interface.

Users can add new students, view student records, update existing information, and delete students without refreshing the page.

---

## 🚀 Features

- ✅ Add new students
- ✅ View all students
- ✅ Update student information
- ✅ Delete students
- ✅ Controlled form inputs
- ✅ REST API integration using Axios
- ✅ Dynamic UI updates
- ✅ Responsive design for mobile, tablet, and desktop
- ✅ Clean and user-friendly interface
